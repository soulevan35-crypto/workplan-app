📌 วิธีตั้งค่าครั้งแรก (ทำครั้งเดียว)
1เปิด script.google.com → New project
2วางโค้ด Apps Script แล้ว Save (ดาวน์โหลดจากไฟล์ที่แนบ)
3Deploy → New deployment → Web app → Execute as: Me / Access: Anyone
4Copy Web app URL มาใส่ด้านล่าง แล้วแชร์ให้ทีม
